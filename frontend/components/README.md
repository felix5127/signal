# Components 文件夹

一旦我所属的文件夹有所变化，请更新我。

**作用**: 存放可复用的 React 客户端组件

**主要文件**:

### 首页核心组件 (BestBlogs风格)
- `resource-card.tsx` - 资源卡片组件（标题、摘要、评分、来源、分类）
- `resource-filter-bar.tsx` - 筛选栏组件（时间/分类/排序/精选开关）
- `pagination.tsx` - 分页组件
- `score-badge.tsx` - 评分徽章组件（支持精选标记）

### 旧版组件 (保留兼容)
- `filter-bar.tsx` - 旧版筛选栏组件
- `daily-news-tab.tsx` - 每日新闻 Tab 页（按日期分组、无限滚动）
- `weekly-report-tab.tsx` - 每周观察 Tab 页
- `deep-research-button.tsx` - 深度研究按钮组件
- `markdown-renderer.tsx` - Markdown 渲染器
