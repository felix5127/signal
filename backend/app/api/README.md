# API - FastAPI 路由层

定义 RESTful API 端点，处理 HTTP 请求并返回响应。

## 文件清单

- `digest.py` - 每日/每周精选汇总 API（GET /api/digest/today, /api/digest/week 等）
- `resources.py` - v2.0 资源管理 API（GET /api/resources, /api/resources/{id}, /api/resources/stats）
- `feeds.py` - RSS Feed 输出 API（GET /api/feeds/rss, /api/feeds/info）
